When to use Flexbox...

In a nutshell flexbox is used to layout and align items in the container when the dimensions are unknown...so it needs to be flexible!

Flex is best used in 1D layouts such as a single row or column.

The Flexbox container...

When flex is applied to a container it gives the ability to alter width or height (and even the order) expanding to fill space or shrinking to prevent overflow to best fill the container.

.container {
  display:flex;
}

To illustrate the flex properties and values the flex container will have 3 flex children named Alpha, Beta and Charlie.

Flex Direction

There are two axis in the flex container, the main axis and cross axis.

flex-direction specifies which axis is the main axis. For example, in a layout in a row the main axis is the x-axis and the cross axis is the y-axis, or if in a column the main axis is the y-axis and the cross axis is the x-axis.

The values available for flex-direction are:

  • flex-direction: row;(default)
  • flex-direction: column;
  • flex-direction: row-reverse;
  • flex-direction: column-reverse;
Flex Wrap

Flex items will try and fit all on one line by default

flex-wrap can be used to change this behaviour

The values available for flex-wrap are:

  • flex-wrap: nowrap (default)
  • flex-wrap: wrap;
  • flex-wrap: wrap-reverse;
