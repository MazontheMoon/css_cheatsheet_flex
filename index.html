What you need to know...

My collection of cheatsheets so far have looked at topics such as the Box Model and Selectors which are about about how the content boxes work and methods to select them, and the looked at Display and Positioning for where to put the selected boxes. Flex is a Display property but deserved a page all to itself.

When to use Flexbox...

In a nutshell flexbox is used to layout and align items in the container when the dimensions are unknown...so it needs to be flexible!

Flex is best used in 1D layouts such as a row or column.

The Flexbox container...

When flex is applied to a container it gives the ability to alter width or height (and even the order) expanding to fill space or shrinking to prevent overflow to best fill the container.
