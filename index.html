When to use Flexbox...

In a nutshell flexbox is used to layout and align items in the container when the dimensions are unknown...so it needs to be flexible!

Flex is best used in 1D layouts such as a single row or column.

The Flexbox container...

When flex is applied to a container it gives the ability to alter width or height (and even the order) expanding to fill space or shrinking to prevent overflow to best fill the container.

.container {
  display:flex;
}

To illustrate the flex properties and values the flex container will have 3 flex children named Alpha, Beta and Charlie.

Flex Direction

There are two axis in the flex container, the main axis and cross axis.

flex-direction specifies which axis is the main axis. For example, in a layout in a row the main axis is the x-axis and the cross axis is the y-axis, or if in a column the main axis is the y-axis and the cross axis is the x-axis.

The values available for flex-direction are:

  • flex-direction: row;(default)
  • flex-direction: column;
  • flex-direction: row-reverse;
  • flex-direction: column-reverse;
Flex Wrap

Flex items will try and fit all on one line by default

flex-wrap can be used to change this behaviour

The values available for flex-wrap are:

  • flex-wrap: nowrap (default)
  • flex-wrap: wrap;
  • flex-wrap: wrap-reverse;
Flex Flow

There is a shorthand for writing flex-direction and flex-wrap which is flex-wrap

So this...

.container {
  display:flex;
  flex-direction: column;
  flex-wrap: wrap-reverse;
}

can instead be written as...

.container {
  display:flex;
  flex-flow: column wrap-reverse;
}
Justify Content

So far we have set the container to use flexbox and to use either a row or column (with or without wrapping). Our next step aligns the flex children on the container main-axis.

The values available for justify-content are:

  • justify-content: flex-start; (default)
  • justify-content: flex-end;
  • justify-content: center;
  • justify-content: space-between;
  • justify-content: space-around;
  • justify-content: space-evenly;

Flex items are packed in starting from the start of the main-axis

Flex items are packed in starting from the end of the main-axis

Flex items are packed in around the center point of the main-axis

The first flex item is flush with the start of the main axis, the last item is flush with the end of the main axis and all remaining items spead evenly between them.

Every flex item has the same amount of space before and after it - this means that the first and last items look like have half the space at the very beginning and end (like a margin that doesn't collapse between siblings).

All flex items distributed along the main axis with equal space between them

Align Items

Similar to justify-content the align-items aligns the flex items along the container but this time on the cross-axis (perpendicular to the main-axis.

The values available for align-items are:

  • align-items: flex-start; (default)
  • align-items: flex-end;
  • align-items: center;
  • align-items: stretch;
  • align-items: baseline;
